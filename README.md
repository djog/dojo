Aan computers vertellen wat ze moeten doen is de kern van programmeren. Wil jij dit ook kunnen? Elke donderdagavond kun je bij de DOJO leren hoe je je eigen interactieve programma's maakt, zoals een Ã©chte game! De jongsten maken programma's voor de desktop PC, de tieners mogen zich wagen aan webgames, apps en andere 
platforms. Voorkennis is niet nodig.

Deze avond is een cursusavond, waarin samen leren programmeren voorop staat. 
Je zult aan de slag gaan met Processing of C++, 
maar je bent ook vrij je eigen weg te kiezen.

 * Dag: Elke donderdagavond, behalve op feestdagen en in de kerstvakantie

Aan computers vertellen wat ze moeten doen is de kern van programmeren. Wil jij dit ook kunnen? Elke donderdagavond kun je bij de DOJO leren hoe je je eigen interactieve programma's maakt, zoals een Ã©chte game! Je kunt programma's leren maken voor PC, maar uiteindelijk ook smartphone of Nintendo DS. Voorkennis is niet nodig.

Deze avond is een cursusavond, waarin samen leren programmeren voorop staat. 
Je zult aan de slag gaan met Scratch, Processing of C++ met Qt, SFML en/of Urho3D, 
maar je bent ook vrij je eigen weg te kiezen.
